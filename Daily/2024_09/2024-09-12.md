# difference and improvment

- **Loss of power results in loss of non-retained PLC data.**  
    PLCs have limited memory for retained data, and the current setup doesn’t cover all the data for some PLCs.  
    PC and SRM are probably fine, but CC and DMS need to be evaluated on a case-by-case basis.  
    _Chilsung code_: Some of data structures should be changed to use smaller data sizes, for example, changing the 22-byte character location to an integer.
    Requires recovery plc and wcs both
    
- **Data lost causing TUCA.**  
    In Chilsung code, we only send TUCA when the operator deletes data. We never delete data without manual intervention
