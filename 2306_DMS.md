- On Aldi Vic, we are only using a single tray size (TRL1) and all locations are double deep
- MS_CONV_POS_GENERIC should be 1, MS_CONV_POS_MAX should be 2 (for AUS DMS, we use specific positions not virtual positions)

- On Aldi Vic, '_MS_AISLE_ASCENDING_LEFT_TO_RIGHT_' should be set to FALSE.
